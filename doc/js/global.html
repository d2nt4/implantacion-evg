buscarCSU - Función para buscar y comprobar datos con AJAX.(baseURL, tabla, valor, campo, idHTML, idInput, textoBase, permitido)

Parameters:
Name Type Description
baseURL string URL base del proyecto.
tabla string Tabla a realizar la consulta.
valor string Valor a contrastar con la base de datos.
campo string Campo de la tabla de la base de datos a comprobar.
idHTML string ID del elemento HTML a mostrar el resultado de la consulta.
idInput string ID del input para colorear el borde de rojo cuando hay un fallo.
textoBase string Resultado obtenido de la consulta.
permitido string Valor para poder modificar una fila si ya existe, es decir, modificar la descripción de una aplicación en la que no modificas su nombre (si no le pasas este valor, el sistema no dejará modificar ya que existe una aplicación con este nombre).
